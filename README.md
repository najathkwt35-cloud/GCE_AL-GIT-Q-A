turned# GIT Seminar Guide — Q&A 📘

A bilingual (English + Tamil) interactive Q&A revision site for GCE A/L GIT students. Built from a Tamil short-notes PDF, converted into easy-to-scan Q&A cards for fast exam prep.

🔗 **Live site:** [gce-al-git-q-a-zyhf.vercel.app](https://gce-al-git-q-a-zyhf.vercel.app)

---

## ✨ Features

- 🇬🇧 / 🌐 **Language toggle** — switch between English and Tamil instantly
- 🃏 **Tap-to-reveal cards** — tap a question card to open the answer
- 🔍 **Search** — quickly find any topic or keyword
- 🎨 **Colour-coded topic pills** — jump straight to a topic
- 📊 **11 Topics · 60+ Q&A cards** — full syllabus coverage, 100% exam-focused

---

## 🛠️ Tech Stack

- HTML5, CSS3, JavaScript (Vanilla)
- Hosted on **Vercel**
- Responsive, mobile-first design

---

## 🚀 Getting Started (Local Setup)

```bash
# Clone the repo
git clone https://github.com/<your-username>/<repo-name>.git
cd <repo-name>

# Open directly in browser
open index.html
```

No build step required — it's a static site, so any local server (Live Server, `python -m http.server`, etc.) works too.

## 📖 Usage

1. Open the site
2. Choose your language (English 🇬🇧 or Tamil 🌐)
3. Use the search bar or colour pills to find a topic
4. Tap any card to reveal the answer
5. Revise fast before your exam 🎯
