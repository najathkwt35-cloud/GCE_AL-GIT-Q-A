A GCE A/L GIT short-notes PDF turned into a fast, tap-to-reveal Q&A revision site. Built for quick exam prep — search or filter by topic, tap a card to flip open the answer.

## ✨ Features

- 🇬🇧 **English** and 🌐 **Tamil** language toggle
- 🔍 Search bar to jump straight to a topic
- 🎨 Colour-coded topic pills for fast filtering
- 🃏 Tap-to-reveal Q&A cards (60+ cards across 11 topics)
- 📱 Mobile-friendly, exam-focused layout
